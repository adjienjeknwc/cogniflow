# 🚀 CogniFlow: AI User Journey & Conversion Predictor

CogniFlow is a full-stack predictive layout optimization tool designed to evaluate screen real estate and conversion potential before writing code. Powered by the Google Gemini computer vision model, CogniFlow maps visual priority, scores visual hierarchy, and introduces an interactive blueprint simulation engine.

---

## 📊 PART 1: The Business & UX Analyst Perspective

### 🎯 The Conversion Optimization Problem
Modern software teams and businesses spend thousands of dollars running live A/B split traffic testing over weeks just to collect baseline metrics on element discoverability. Poor layout choices—such as placing an essential call-to-action (CTA) below distracting background illustrations—result in severe drop-offs and lost business revenue before layout inefficiencies are ever tracked.

### 🧠 The Solution: Visual Gaze Tracking & Predictions
CogniFlow maps visual saliency immediately upon mockup creation. It targets visual drop-offs by computing element weight balances across a layout.

* **Conversion Score Meter (0-100):** A custom layout health index that scales downward if distracting artwork crowds out critical commercial conversion goals.
* **The Roadmap Gaze Tracker (#1, #2, #3):** Outlines chronological user fixation sequences based on sizing, background isolation, and contrast weighting.
* **Friendly Design Critique:** Translates layout coordinate maps into actionable design recommendations written in human-readable terms.

## 🛠️ The Architectural Stack

* **Frontend User Interface:** React.js, Tailwind CSS, Framer Motion (Concentric animation loader sequencing), HTML5 Canvas context APIs.
* **Backend Server Mesh:** Node.js, Express.js microservice router, Multer stream handling pipeline.
* **AI Engine Handshake:** `@google/generative-ai` integration processing inline binary buffer payloads securely.
* **Production Hosting Environments:** Render cloud services (Independent Web Service and Static Site distributions).
